## Quick Start

Install and start using AI components in your Next.js project in just a few steps.

### Requirements

Before installing the library, make sure your project includes:

- _Next.js 14+_
- _React 18+_
- _Tailwind CSS_

If you are installing through CLI follow shadcn [documentaion](https://ui.shadcn.com/docs/installation)
